Timer, Stopwatch, Clock

This feature is used for a variety of reasons - perhaps you want to time how long it takes to do something, or you want to set a timer, so you can remind yourself of some task on your to-do list. Maybe you need to find out what your timezone is, ASAP. Whatever it is, you can use this feature to help you!
The Timer function is used to set timers! With a simple and clean interface, all you need to do is set your time and hit start. You can also pause, play, and reset the timer whenever you like.
To keep track of time, use the Stopwatch function! Just click start and watch the time fly. If necessary, you can pause/unpause the timer and reset it as well.
Use the Clock function to see the day of the week, the current date, the local time, and the timezone that you're in!
